1. Установите расширение Live Server

    1. Откройте VS Code.
    2. Перейдите в раздел Extensions (Расширения):
        - Нажмите на иконку с квадратами в левой боковой панели (или используйте сочетание клавиш Ctrl+Shift+X / Cmd+Shift+X на Mac).
    3. В строке поиска введите Live Server.
    4. Найдите расширение Live Server от Ritwick Dey и нажмите Install (Установить).

2. Подготовьте ваш проект

    Убедитесь, что у вас есть HTML-файл (например, index.html), который вы хотите открыть через Live Server.
    Сохраните файл в папке проекта (например, C:\Projects\MyWebsite\).

Пример структуры:
text
MyWebsite/
├── index.html
├── styles.css
└── script.js

3. Запустите Live Server

    Откройте файл index.html в VS Code:
        Щёлкните по файлу в проводнике (Explorer) слева.
    Запустите сервер одним из способов:
        Способ 1: Щёлкните правой кнопкой мыши внутри открытого index.html и выберите Open with Live Server.
        Способ 2: Нажмите на кнопку Go Live в правом нижнем углу окна VS Code (появляется после установки расширения).
    Браузер автоматически откроется с вашим файлом по адресу, например, http://127.0.0.1:5500/index.html.

4. Работа с Live Server

    Автообновление: При изменении и сохранении файлов (index.html, CSS, JS) страница в браузере автоматически обновится.
    Проверка: Убедитесь, что ваш код работает корректно (например, отправка формы на сервер FastAPI).

5. Остановка Live Server

    Чтобы выключить сервер, нажмите на кнопку Port: 5500 (или другой порт) в правом нижнем углу VS Code — она изменится с "Go Live" на номер порта после запуска.
    Либо закройте VS Code.

Полезные настройки (опционально)

    Откройте настройки расширения:
        Перейдите в File > Preferences > Settings (или Ctrl+,).
        Найдите Live Server через поиск.
    Настройте параметры:
        liveServer.settings.port: Измените порт (по умолчанию 5500), если он занят.
        liveServer.settings.root: Укажите корневую папку, если ваш index.html не в корне проекта (например, /public).
        liveServer.settings.host: Оставьте 127.0.0.1 для локальной работы.

Пример запуска с вашим проектом

    Сохраните ваш HTML-код из предыдущих сообщений как index.html в папке проекта.
    Убедитесь, что сервер FastAPI запущен (python main.py на http://localhost:8000).
    Откройте index.html в VS Code и выберите Open with Live Server.
    В браузере откроется страница, и вы сможете отправлять заказы через форму — данные будут сохраняться в PostgreSQL.

Возможные проблемы и решения

    Порт занят: Если порт 5500 занят, Live Server выдаст ошибку. Измените порт в настройках или завершите другой процесс (netstat -a -o в Windows или lsof -i :5500 в Linux/Mac для проверки).
    CORS: Если запросы к FastAPI не проходят, проверьте настройки CORS в main.py (allow_origins) — добавьте http://127.0.0.1:5500.
